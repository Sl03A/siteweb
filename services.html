Questions Fréquentes

Tout ce que vous devez savoir avant de commander un service

Quels sont les délais de réalisation moyens ?

Les délais varient selon le type de projet :

  • Montage vidéo : 3-7 jours selon la longueur
  • Site vitrine : 1-3 semaines
  • Site e-commerce : 2-4 semaines
  • Design logo : 5-10 jours ouvrables
  • Application web : 1-3 mois

Un planning détaillé est toujours fourni avant le début du projet.

Proposez-vous des révisions après livraison ?

Oui ! Tous mes services incluent des révisions gratuites :

  • Pack Basic : 2 révisions
  • Pack Standard : 3 révisions
  • Pack Premium : Révisions illimitées

Nous travaillons en étroite collaboration jusqu'à ce que vous soyez entièrement satisfait du résultat.

Comment se passe le paiement ?

Plusieurs options de paiement sont disponibles :

  • Petits projets : 100% à la livraison
  • Projets moyens : 50% d'acompte, 50% à la livraison
  • Gros projets : 30% à la commande, 40% à mi-parcours, 30% à la livraison

Paiement par virement bancaire, PayPal, carte bancaire ou cryptomonnaies (Bitcoin, Ethereum).

Fournissez-vous l'hébergement pour les sites web ?

Oui, je propose différentes options d'hébergement :

  • Hébergement basique : 1 an offert avec tout développement web
  • Hébergement premium : Serveur optimisé avec SSL, backups automatiques
  • Maintenance : Mises à jour, sécurité, sauvegardes

Je peux également configurer l'hébergement sur votre propre serveur si vous préférez.

Travaillez-vous avec des entreprises à l'étranger ?

Absolument ! Je travaille régulièrement avec des clients internationaux. Voici comment cela se passe :

  • Communication : Français, Anglais, Espagnol
  • Fuseaux horaires : Adaptabilité totale
  • Paiement : Devises multiples acceptées
  • Outils : Zoom, Slack, Trello, etc.

La distance n'est pas un problème avec les outils de collaboration modernes.

Vous ne trouvez pas ce qu'il vous faut ?

Chaque projet est unique. Je crée des solutions sur mesure adaptées à vos besoins spécifiques.
